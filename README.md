### Steps

1. Open the `GK07_Demo` folder
2. Right-click `GK07_Demo.uproject`
3. Select **Generate Visual Studio project files**
4. Double-click `GK07_Demo.uproject`
5. Wait for shaders to compile
6. Open map **Demo** (folder: `Midterm`) and press **Play**

### Troubleshooting

If the project fails to open:

* Delete these folders inside `GK07_Demo`:

  * `Binaries`
  * `Intermediate`
  * `Saved`
* Then repeat from step 3
